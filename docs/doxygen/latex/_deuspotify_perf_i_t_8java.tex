\doxysubsection{Detailed Description}
Performance integration tests for Deuspotify using JUnit\+Perf. 

This class defines performance benchmarks for key service methods in the Deuspotify application. It uses the JUnit\+Perf library to simulate concurrent load and validate performance metrics such as throughput and latency percentiles.

The results are saved in an HTML report under {\ttfamily target/reports/perf-\/report.\+html}.
